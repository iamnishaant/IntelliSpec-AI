# Library Management System

## Actors
1. **Librarian**: Manages the books and user accounts.
2. **Member**: Can borrow and return books.

## Features
1. **Borrow Book**: A member selects a book and borrows it if it's available.
2. **Return Book**: A member returns a book they previously borrowed.
3. **Add Book**: The librarian adds a new book to the library system.
4. **Register Member**: The librarian registers a new member.

## Process
1. When a book is borrowed, the system updates the inventory.
2. When a member returns a book, the status is set back to available.
